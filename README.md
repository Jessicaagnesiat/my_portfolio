# Streamlit Data Portfolio

This project is an interactive data portfolio built using Streamlit.

## Features
- Interactive dashboard and data visualization
- Sidebar navigation
- KPI metrics and charts
- Deployed using Streamlit Cloud

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Plotly
